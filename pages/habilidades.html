MINHAS HABILIDADES.

HTML

Iniciei meus estudos em HTML aprendendo a estruturar páginas web, evoluindo para o uso de elementos semânticos e organização correta do conteúdo. Com o tempo, passei a construir páginas mais bem estruturadas e organizadas, aplicando boas práticas de marcação.

CSS

Meu aprendizado em CSS começou com estilizações básicas e evoluiu para a criação de layouts mais organizados e responsivos. Ao longo do desenvolvimento, passei a utilizar media queries e técnicas de adaptação para diferentes dispositivos, aprimorando a apresentação visual das páginas.

JavaScript

Comecei com os conceitos fundamentais da linguagem e evoluí para a criação de scripts mais dinâmicos. Com o tempo, desenvolvi a capacidade de manipular elementos da página, criar interações e implementar lógicas mais estruturadas em projetos simples.

Git & GitHub

Controle de versão com Git e colaboração via GitHub, utilizando branches, pull requests e boas práticas de versionamento para projetos organizados e eficientes.

Angular

Conhecimentos em Angular para desenvolvimento de interfaces web dinâmicas e responsivas, utilizando componentes, serviços, roteamento, consumo de APIs REST e boas práticas de desenvolvimento frontend.
